== Description ==

Serverclub.Digital is a popular SMS gateway for Sri Lanka which you can use to send transactional or bulk SMS to your customers. This plugin allows you to integrate Serverclub.Digital SMS service into your WooCommerce store.

= Features =

* Send SMS to your customers automatically for any WooCommerce order status. You can let them know if you received the order, when they make the payment, you process the order, when the order is ready, when order is shipped or any custom event you define.
* Receive an SMS to one or multiple admin mobile numbers when your store got a new order.
* Send SMS when you add a customer note.
* You can customize the content of every automated SMS. 


== Installation ==

= Automatic Installation =

1. In the dashboard navigate to `Plugins > Add New`
2. Search for `Serverclub.Digital SMS for WooCommerce`
3. Locate the `Serverclub.Digital SMS for WooCommerce` by `ServerClub.LK (Private) Limited` and click `Install Now`
4. After installation, click on `Activate Plugin` to activate

= Manual Installation =

1. Download the zip file using the `Download` button above in this page
2. In the dashboard navigate to `Plugins > Add New`
3. Click on `Upload Plugin`
4. Click `Choose File` then select the zip file you downloaded
5. Click `Install Now`, wait for the installation and click `Activate Plugin`

= Enabling the Landing Page =

1. In the dashboard, navigate to `WooCommerce > Settings`
2. Go to `Serverclub.Digital SMS` tab
3. Fill all options required, and save the settings

1, 2, 3: You're done!

== Frequently Asked Questions ==

= How to sign-up for an account? =
Go to our website https://sms.serverclub.digital/ and register for an account.

= Can this plugin send SMS? =
Yes, if you are registered Serverclub.Digital user, you can integrate this plugin with Serverclub.Digital and trigger SMS automatically for WooCommerce order status updates and order note updates as well.
